
# 🚀 Cigg-App – DevOps Showcase Project

## 📌 Overview

**Cigg-App** is an AI-generated application where the core codebase is created using AI tools, while the **complete DevOps lifecycle — from development to production deployment — is designed and implemented by me**.

This project demonstrates real-world DevOps practices including CI/CD, containerization, orchestration, monitoring, and cloud deployment.

---

## 🎯 Objective

* Demonstrate end-to-end DevOps pipeline implementation
* Showcase hands-on experience with production-grade tools
* Build and deploy a scalable, observable application

## 🧱 Architecture Overview

* **Local Development & Testing**

  * Kubernetes cluster using Minikube
  * Used for validating deployments before cloud rollout

* **CI/CD Pipeline**

  * GitHub Actions automates:

    * Build
    * Docker image creation
    * Push to container registry
    * Deployment updates

* **Containerization**

  * Docker ensures environment consistency

* **Orchestration**

  * Kubernetes handles:

    * Deployment
    * Scaling
    * Service exposure

* **Production Deployment**

  * Hosted on AWS EKS (Elastic Kubernetes Service)
  * Designed for scalability and high availability

* **Monitoring & Observability**

  * Prometheus for metrics collection
  * Grafana for visualization dashboards
  * Tracks application and cluster performance

* **Infrastructure as Code (Upcoming)**

  * Terraform will automate provisioning of:

    * EKS cluster
    * Networking
    * Cloud resources

---

## 🔄 CI/CD Workflow


Code Push → GitHub Actions → Build Docker Image → Push to Registry → Deploy to Kubernetes (EKS) → Monitor via Prometheus & Grafana
